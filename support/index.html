Support

Have a question about something you can't find in the documentation? Try one of the following resources.

Code of Conduct

Folktale aims to be a friendly, welcoming, and inclusive community, so we have a few rules in place to ensure that. In short: be respectful, but you can see the full details of expectations and enforcement on the Code of Conduct page.

Gitter Chat

There's a chatroom on Gitter where you can have your questions quickly-ish answered.

FAQ

Answers to some very common questions that people have about Folktale.

Issue tracker

If you've found a bug, something missing in the docs, or just wish Folktale did something it doesn't do yet, the GitHub issue tracker is the place to go.
